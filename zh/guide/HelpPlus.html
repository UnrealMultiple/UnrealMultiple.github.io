HelpPlus 更好的帮助

  • 作者: Cai
  • 出处: 此仓库
  • 修复/help指令无法使用的问题
  • 为/help添加简短提示
  • 为/help <命令> 可以看见更详细的命令帮助

指令

/help <页码> 查看命令列表
/help <命令> 查看命令的详细帮助

配置

配置文件位置：tshock/HelpPlus.json

{
  "简短提示开关": true,
  "简短提示对应": {
    "user": "用户管理",
    "login": "登录",
    "logout": "登出",
    "password": "修改密码",
    "register": "注册",
    "accountinfo": "账号信息",
    "ban": "封禁",
    "broadcast": "广播",
    ......
    "deal": "交易",
    "igen": "快速构建",
    "relive": "复活NPC",
    "bossinfo": "进度查询"
  }
}
